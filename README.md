<h1 align="center">XBlockFire</h1>

  <p align="center">
    一款基於 Minecraft Bedrock Edition 所打造的開源第一人稱射擊（FPS）遊戲插件。透過 Script API 打造遊戲系統，結合行為&資源包設計槍枝物品，並引進最經典的爆破模式，讓玩家能夠在 Minecraft 的世界中體驗第一人稱射擊的樂趣。
